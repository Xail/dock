==Отображение запущенный приложений==
Под приложениями понимаются приложения, имеющие окна.
Нужно получить список окон верхнего уровня и их владельцев.

Для получения списка окон будем использовать процедуру EnumWindows
Она получит хэндлы окон. Отображать приложения на форме будем в
виде кнопок с названиями приложений.

Итого
     Хэндл - Приложение - Кнопка

Модель:
Кнопка  - Процесс - Окно
ExeName - PID     - Handle
Кнопка
- Список процессов
- - список окон - подменю

Представление
Панель кнопок
- Кнопка
- - Счетчик окон

Создается панель кнопок. Она запускает перечисление видимых окон.
Окна группируются в процессы,
процессы группируются в кнопки
